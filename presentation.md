Title: Scikit Functional Data Analysis Package in Python

Abstract: Presentation breakdown of a scikit (Python machine learning library) package for functional data analysis in Python. Functional data analysis views observations as functions of variables; this allows for continuous, smooth graphs which are beneficial for visualizing numerical computation.
